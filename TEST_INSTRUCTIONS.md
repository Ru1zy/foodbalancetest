# Инструкция для тестирования Telegram Login

## Тест 1: Проверка кук telegram.org
1. Открой DevTools (F12)
2. Application → Cookies
3. Проверь, есть ли куки от домена `.telegram.org`
4. Если есть - удали их все
5. Обнови страницу
6. Попробуй нажать на Telegram Login Widget

**Ожидаемый результат:** Если виджет просит номер телефона - значит проблема в отсутствии кук telegram.org

## Тест 2: Проверка через telegram.org
1. Открой https://telegram.org в том же браузере
2. Нажми "Sign in" в правом верхнем углу
3. Авторизуйся через QR-код или номер телефона
4. Вернись на сайт FoodBalance
5. Попробуй Telegram Login Widget

**Ожидаемый результат:** Виджет должен работать мгновенно без запроса номера

## Тест 3: Проверка Mini App
1. Открой бота @fooddevtestbot в Telegram
2. Нажми кнопку Menu (☰) внизу
3. Должно открыться приложение
4. Проверь консоль - должно быть "TMA AUTH PAYLOAD: ..."
5. Вернись на сайт в браузере
6. Проверь, сохранился ли профиль

## Вывод
Если Тест 1 показывает запрос номера, а Тест 2 работает - значит Telegram Login Widget работает ТОЛЬКО для пользователей с активной сессией на telegram.org.

Это НЕ баг в коде, это ограничение самого Telegram Login Widget.
